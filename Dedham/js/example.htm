This script was tested with the following operating systems and browsers:

Windows XP:Internet Explorer 6, Netscape Navigator 4 + 7, Opera 7, Firefox 1
Mac OS X:Internet Explorer 5, Safari 1*

*NOTE: With Safari 1 (Mac OS X) performance is very poor, i.e. ticker speed is too slow.

If you use another browser or operating system, this script may not work for you - sorry.

Usage

Insert your ticker entries (var tickerEntries) and adapt the configuration to your needs. Then insert this line into your HTML-body where you want the ticker to appear: 
<script language="JavaScript" src="ticker.js"></script>
The actual width of the ticker DIV depends on the text length of your ticker messages. You can of course set the visible ticker area to any size you wish, but this has no effect on the actual width. If the actual width is bigger than the width of your browser window, a horizontal scrollbar will appear. You can avoid this by putting the page with the ticker script into a frameset with scrollbars turned off.
